Application Web Commautaire

Présentation :

Je vous présente une application web que j'ai réalisée dans le cadre d'un projet universitaire. L'objectif était de créer une plateforme communautaire en PHP où les étudiants pourraient créer une session personnelle et y déposer leurs projets.

Une fois connecté, les utilisateurs peuvent consulter leurs statistiques à l'aide d'un tableau affichant les moyennes dans les trois principaux thèmes de la formation. Ils ont également la possibilité d'ajouter, modifier ou supprimer des projets.

Une fonctionnalité importante concerne les traces associées à chaque projet lors du processus de création. Une page dédiée permet d'ajouter, remplacer ou supprimer ces traces.

Pour le développement de ce projet, j'ai utilisé le logiciel Visual Studio Code. J'ai programmé les différentes pages en HTML, CSS et Javascript, et j'ai utilisé le PHP et le SQL pour gérer toutes les données du site.
