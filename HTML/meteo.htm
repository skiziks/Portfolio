Site météo

Présentation :

je vous présente un site que j'ai crée entièrement qui consiste à voir les informations météorologique d'une ville, la ville par défault est Paris, mais une barre de recherche est mise en place en haut à droite pour rchercher n'importe quelle ville dans le monde pour avoir ses informations météorologique.

Technologiquement, ce site est développer en React JS en interrogeant l'API de météo open weather map qui me permet d'obtenir les informations météorologique de l'intégralité des villes de la planète selon ce que l'utilisateur choisi.

J'ai réaliser ce projet personnel dans un premier but de m'améliorer à l'utilisation du framework React JS surtout avec l'utilisation d'une API, mais dans un second temps pour que ce soit utile pour moi car je trouvais mon site personnelement plus agréable à consulter, c'était donc un souhait de ma part d'avoir accès aux informations météorologique que je souhaitais avec une interface plus agréable.
