A propos du projet :

Dans le cadre d'un projet de fin d'année que nous devions réaliser en équipe, nous avions comme mission principale d'imaginer une campagne de sensibilisation des jeunes aux enjeux du numérique sur le thème de l'humain alliant les années 70.

Pour cela, nous avons choisi de partir sur l'addiction aux réseaux sociaux, car c'est un fléau majeur sur les relations humaines et dont l'homme, de manière générale, a complètement changé sa manière de vivre depuis leur apparition.

Aspect technique :

Dans le groupe, je me suis occupé de la partie développement web accompagné d'un collègue, nous étions donc 2 pour le développement de ce site.

Pour le réaliser nous avons utilisé HTML, CSS ainsi que JavaScript pour certaines parties techniques telles que l'utilisation de Chart JS, afin de montrer des statistiques de consommation plus concrètement à l'aide d'un graphique, de menu dropdown qui apparaisse lors que l'on scroll progressivement sur le site pour nous donner des informations intéressantes et enfin pour la création d'un quizz complet sous forme de vrai ou faux de 5 questions pour tester ses connaissances sur l'addiction aux réseaux sociaux.

Travail en équipe :

Comme je l'ai dit dans le premier paragraphe, ce projet a été réalisé en équipe, ce qui implique une méthodologie concrète pour réussir le travail au mieux.

Pour le développement, nous avons utilisé Git et GitHub afin de se coordonner sur certaines parties du site différemment à l'aide de branches créées séparément, mais également afin d'obtenir toute la puissance de Git avec la gestion des versions et de garantir un développement optimal sans risque de problèmes critiques, car nous aurions pu à tout moment récupérer une ancienne sauvegarde.

Du point de vue du groupe de manière générale, nous avons utilisé un outil qui s'appelle Trello et qui permet de coordonner les missions de chacun pour un projet afin que toute l'équipe sache quoi faire pour faire pour faire avancer le projet plus vite. Dans mon cas, j'ai trouvé très utile pour savoir l'avancée des autres membres du groupe et avoir un aperçu de l'avancée globale du projet.
